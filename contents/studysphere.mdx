# 📚 Study Sphere

> A slick, animated Ed-Tech website built with **Next.js** – because learning should look cool too. 🚀✨

---

## 🧠 What is Study Sphere?

**Study Sphere** is a modern, animated educational platform landing page crafted using the latest web tech.  
Think of it as the digital face of your favorite ed-tech startup – clean UI, buttery-smooth transitions, and Gen-Z vibes throughout.

---

## 🛠️ Tech Stack

- ⚛️ **Next.js** – React on steroids  
- 🎨 **Tailwind CSS** – Utility-first CSS that slaps  
- 🧠 **Framer Motion** – For crisp animations that feel alive  
- 💅 **ShadCN UI** – Pre-built UI components that actually look good  
- 🔤 **TypeScript** (optional) – Because type safety is peace of mind  

---

## 🚀 Features

- 🎬 Smooth, scroll-based animations  
- 🎯 Responsive and mobile-friendly layout  
- 🧩 Modular component structure  
- 💡 Perfect for Ed-Tech startups, course creators, or portfolio projects
