
# 🪙 **Solana Token Launchpad**

**Tech Stack:** React, Tailwind CSS, Solana Web3.js, SPL Token, ShadCN UI

**Description:**
A fully functional token creation platform built on Solana Devnet. Users can input token details (name, symbol, metadata URI, and supply) and mint their own SPL tokens directly into their wallet using the Solana Wallet Adapter.

**Key Features:**

* 🔗 Wallet connect using Solana Wallet Adapter (e.g., Phantom)
* 🪙 Token creation & minting via `@solana/spl-token`
* 🧠 Metadata setup via `@solana/spl-token-metadata`
* 🎨 Responsive UI with a custom color palette: #000000, #F4E7F2, #9B7197
* 🧾 JSON metadata preview using a custom `CodeBlock` component
* ⚙️ Fully on-chain logic executed in a single transaction

**Outcome:**
With one click, users can generate a new token, initialize metadata, and mint it to their Devnet wallet — ideal for demo tokens or test deployments.
